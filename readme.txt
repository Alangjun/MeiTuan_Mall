flex-wrap: wrap 设置换行属性
flex-direction:row 设置排列方式为横向排列
align-content 多行在交叉轴（非主轴上的对齐方式）
align-items 规定所有的子项在布局中对齐方式，是start|end|center|strech
strech: 每个item之间和行与行之间存在默认的距离值


默认的主轴是column

顺着主轴
just-content:属性定义了浏览器如何顺着父容器的主轴元素以及周围的空间，默认为flex-start

顺着侧轴
alignItems 'flex-start', 'flex-end', 'center', 'stretch'
与justity-content相同的方式在侧轴方向上就爱将当行的元素进行对齐  默认为stretch
